# Border

## Border Radius

The border radius is a set of predefined values that can be used to create rounded corners on elements. The border radius values are defined in the `assets/style/variables/_border.scss` file.
The values are inspired by the [Tailwind CSS](https://tailwindcss.com/docs/border-radius) library and [Open Props](https://open-props.style) project.
