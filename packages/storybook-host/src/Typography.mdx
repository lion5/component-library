# Typography

The typography system is based on the `rem` unit, which is relative to the root font size. This allows for a consistent and scalable design system. The typography system is based on the following variables: font family, font size and font weight.
Most of the typography system is inspired by the [Open Props](https://open-props.style/#typography) and [Tailwind CSS](https://tailwindcss.com/docs/font-family) projects.

## Font Family
The font family system is based on the generic families, which are `sans-serif`, `serif` and `monospace`. The specialized family `display` is used for special cases. If prop not set it defaults back to `sans-serif`. The generic families are used for general text, while the specialized families are used for specific elements.
